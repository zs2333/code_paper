# 程序流程结构

## 执行语句

### 1.单行格式if语句

if(条件){条件满足执行的语句}

```c++
int main()
{
	//选择结构 单行if语句
	int score = 0;
	cout << "请输入你的分数: " << endl;
	cin >> score;
	if (score > 600) {
		cout << "你的分数大于600" << endl;//注意if (score > 600)这里后面不要加分号，直接跟执行语句就可以，加了分号就失去了判断功能
	}
	if (score <= 600) {
		cout << "你的分数不大于600" << endl;
	}

	system("pause");

	return 0;
}
```

### **5.三目运算符**

作用：实现简单的判断

语法：表达式1 ?  表达式2 : 表达式3

含义：

如果表达式1的值为真，执行表达式2，并返回表达式2的结果；

如果表达式1的值为假，执行表达式3，并返回表达式3的结果；

```c++
int main(){
	//三目运算符
	int a = 30;
	int b = 40;
	int c = 50;
	c = (a > b ? a : b)=100;
	cout << "c=" << c << endl;
	//在C++中。三目运算符返回的是变量，可以继续赋值
	c = 20;
	cout << "b=" << b << endl;

	system("pause");

	return 0;
}
```

### **6.switch语句**

作用：执行多条件分支语句

```c++
switch (表达式) 
{
	case 结果1: 执行语句; break;
	case 结果2: 执行语句; break;
		...
	default: 执行语句; break;
}
```

案例：

```C++
int main(){
	//switch 语句
	/*给电影打分
	* 10~9 经典
	* 8~7 非常好
	* 6~5 一般
	* 5以下 烂片
	*/

	//提示用户给电影打分
	cout << "请输入你给电影的打分：" << endl;
	int score = 0;
	cin >> score;
	switch (score)
	{
	case 10: 
		cout<<"经典电影"<<endl; 
		break;//退出当前分支;
	case 8:
		cout << "电影挺好" << endl; 
		break;
	case 6:
		cout << "电影不错" << endl;
		break;

	default: cout << "电影垃圾" << endl;
		break;
	}
	system("pause");

	return 0;
}
```

**if和switch的区别？**

switch的缺点，判断的时候只能值整型或者字符型，不可以是一个**区间**

switch的优点，结构清晰，执行效率高

**注意1：**switch语句中的表达式类型只能是整型或者字符型

**注意2：**case中如果没有break，程序就会一直向下执行

**注意3：**与if语句比，对于多条件判断时，switch的结构清晰，但不能判断区间



## 循环结构

### 1.while循环语句

作用：满足循环条件，执行循环语句

语法：while(循环条件){循环语句}

解释：只要循环条件的结果为真，就执行循环语句

<img src="../../assets/image-20220820171429748.png" alt="image-20220820171429748" style="zoom:67%;" />

```C++
int main(){
	//while (循环条件){循环语句}
	int num = 0;
	while (num < 10) 
    {
		cout << "num = " << num << endl;
        num++;
	}

	system("pause");

	return 0;
}
```

**while+if 案例：**

#### 1.1 do ... while 循环语句

作用：满足循环条件，执行循环语句

语法：do{循环语句}while(循环条件)；

**注意：**与while的区别在于do...while会先执行一次循环语句，再判断循环条件

<img src="../../assets/image-20220820213839589.png" alt="image-20220820213839589" style="zoom: 67%;" />

```python
int main(){
	//do while 语句
	//在屏幕中输出0到9这是个数字
	int num = 0;
	do
	{
		cout << "num:" << num << endl;
		num++;
	} while (num<11);

	system("pause");

	return 0;
}
```

**案例：计算水仙花数**

三位数，个位 十位 百位，的三次幂之和等于该数，这样的数称为水仙花数。

如1^3+5^3+7^3=157

```C++
int main(){
	//do while 语句
	//在屏幕中输出0到9这是个数字
	int num = 100;

	do
	{
		int a = 0;
		int b = 0;
		int c = 0;
		a = num % 10;
		b = (num / 10) % 10;
		c = num / 100;
		
		if (num == a * a * a + b * b * b + c * c * c) {
			cout << num << "是水仙花数" << endl;
		}
		num++;
	} 
	while (num<1000);

	system("pause");

	return 0;
}
```

这里如果次幂很高不方便的话，可以用pow(a,b)函数，表示a的b次幂

代码的if条件句那部分可以改成

```C++
if (num ==pow(a,3) + pow(b,3) + pow(c,3)) {
    cout << num << "是水仙花数" << endl;
}
```

### 2.for 循环语句

作用：满足循环条件，执行循环语句

语法：for（起始表达式；条件表达式；末尾循环体）{循环语句；}

<img src="../../assets/image-20220821144809475.png" alt="image-20220821144809475" style="zoom: 50%;" />

**案例：敲桌子**

1~100个数字中找到 7的倍数、个位有7、或者十位有7，打印”敲桌子”，其他正常打印

```C++
int main(){
	//for 循环
	for (int i = 1;i<=100;i++)
	{
		if (i % 7 == 0 || i % 10 == 7 || i / 10 == 7 )
		{
			cout << "敲桌子" << endl;
		}
		else {
			cout << i << endl;
		}
	}

	system("pause");

	return 0;
}
```

### 4.跳转语句

#### 4.1 break

作用：跳出选择结构或者循环结构

用法：

1.出现在switch条件语句中，作用是**终止case**并跳出switch

2.出现在循环语句中，作用是跳出**当前的**循环语句

3.出现在嵌套循环中，跳出**最近的**内层循环语句

3.代码

break放在外层

```c++
int main(){
	//break的使用时机
	//3.出现在嵌套循环语句中
	for (int i=0;i<10;i++){
		for (int j=0;j<10;j++){
			cout << "* ";
		}
		break;
		cout << endl;//这句话直接不执行
	}
	system("pause");

	return 0;
```
